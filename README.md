# EcoForecast App

EcoForecast is a web app that allows users to visualize and analyze future ecological changes in the sagebrush habitat with hundreds of climate change scenarios. Data generated using [STEPWAT2](https://github.com/DrylandEcology/STEPWAT2) on Yale's cluster compute and is built using [Google Earth Engine](https://developers.google.com/earth-engine). 

Made with researchers from [Yale](https://environment.yale.edu/), the [USGS](https://www.usgs.gov/), [Marshall](https://www.marshall.edu/), and [Utah State](https://www.usu.edu/).
